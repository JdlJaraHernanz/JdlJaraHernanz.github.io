This webpage has been designed by my own, looking for help in forums and learning what there is to know about the basics of front-end web design. I do not claim myself to be an advanced user in these languages (specially in JS, I barely got that to work with some simple functions). If anything, I see myself as an amateur, someone who just enjoys doing things on the computer on a very standard level (even if sometimes they end up looking quite silly). But I hope that this webpage can show both the capabilities of web design (not all, since back-end is quite large and offers even more) and how simple people like me can actually do their own little page just by writing and understanding the code.


Thus, if you check the code (may god - or which ever diety you believe - bless your soul), please understand that it comes from an amateur. And if you have some advice or any suggestions on how I should have done some sections of the page, let me know, I am always keen on improving my skills.


To conclude, a shoutout to GitHub Pages which I am using to host this webpage, great if you look to make your little space on the internet.
